# ✨ Smart_Chart
**요약**
- 환자와 의사를 위해 빠른 일처리를 돕는 smart_chart 서비스 입니다.
- 환자는 병원 예약, 진료비 내기, 기본 건강체크, 실시간 진료상담을 이용할 수 있습니다.
- 의사는 예약/진료 관리, 환자 대기 관리, 매출관리, 실시간 진료상담을 이용할 수 있습니다.
